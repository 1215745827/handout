`查老师的讲义` 

有些时候想学习技术，找了视频发现太长，找了书籍发现太"厚"，找了技术博客发现要么太简单要么太难。

查老师在教学领域里待了这些年，比不了廖老师和各博客大牛，但希望自己的学习和讲义总结能够让一个"小白"受用，也尽可能通过文字让你感受到学习兴趣。

如果查老师的这些讲义和杂文帮到你了，可以猛灌查老师几杯 Java :coffee: 。
